### Mesto

<h1 aline="center">Проект "Место"</h1>

Подготовил студент курса Яндекс Практикум "Веб-разработчик плюс" Тарасенко Марина в рамках изучения темы "Адаптивная верстка" и "Интерактивные элементы"
